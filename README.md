== Description ==


Quickly compare pages or posts on one site with another, checking for missing URLs and missing SEO tags.


== Installation ==


1. Upload the 'ingeni-seo-meta-compare' folder to the '/wp-content/plugins/' directory

2. Activate the Ingeni SEO Meta Compare plugin through the 'Plugins' menu in WordPress

3. Select 'SEO Meta Compare' from the 'Tools' menu in Wordpress.

4. Add the target domain name to compare, whether you are comparing pages or posts.
